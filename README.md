# multi-attachment-inserter
Insert multiple attachments and camera photos. Supports mobile and desktop.
在 Obsidian 中插入多个附件和相机照片。支持移动端和桌面端。


## 📝 简介

Multi-Attachment Inserter 是一个功能强大的 Obsidian 插件，帮助你快速插入多个附件和相机照片。无论你使用桌面端还是移动端，都能轻松地将图片、文档等文件插入到笔记中。

---

## ✨ 功能特性

| 功能 | 描述 | 平台支持 |
|------|------|----------|
| 📎 批量插入附件 | 支持一次选择多个文件并插入到笔记 | 全部平台 |
| 📷 相机拍照 | 直接调用相机拍照并插入笔记 | 移动端 |
| 💾 保存到相册 | 拍照后自动保存到系统相册 | 移动端 |
| 📁 自定义路径 | 设置附件保存的目录路径 | 全部平台 |
| 📏 文件大小限制 | 可配置单个文件大小上限 | 全部平台 |
| 🎨 编辑器工具栏 | 在编辑区显示快捷按钮 | 全部平台 |
| 🌐 多语言支持 | 支持 9 种语言 | 全部平台 |

---

## 📱 支持平台

- ✅ Windows
- ✅ macOS
- ✅ Linux
- ✅ Android
- ✅ iOS

> **注意**：相机拍照和保存到相册功能仅在移动端可用（Android/iOS）。

## 📖 使用方法

### 插入附件

#### 方法一：功能区图标

点击左侧功能区的📎图标，弹出文件选择器，选择文件后自动插入。

#### 方法二：命令面板

1. 按 `Ctrl+P`（Windows/Linux）或 `Cmd+P`（macOS）
2. 输入 "插入附件"
3. 选择 "Insert Multiple Attachments"

#### 方法三：编辑器工具栏

在编辑器底部找到附件按钮，点击即可选择文件。

#### 方法四：拖拽（桌面端）

直接将文件拖入编辑器窗口。

---

### 拍照插入（移动端）

#### 方法一：功能区图标

点击左侧功能区的📷图标，打开相机。

#### 方法二：命令面板

1. 按 `Ctrl+P`
2. 输入 "拍照"
3. 选择 "Take Photo & Insert"

#### 方法三：编辑器工具栏

在编辑器底部点击相机按钮。

---

### 拍照模态框使用

1. 点击"开始拍摄"启动相机
2. 拍摄第一张照片
3. 可选择：
   - **重拍**：删除最后一张，重新拍摄
   - **继续拍**：保留当前照片，继续拍摄下一张
   - **确定**：完成拍摄，将所有照片插入笔记
   - **取消**：取消操作，不插入任何照片

## ⚙️ 设置说明

在 Obsidian 设置中找到 **Multi-Attachment Inserter** 进行配置：

| 设置项 | 说明 | 默认值 |
|--------|------|--------|
| 照片保存路径 | 照片/附件在库中的保存目录 | 使用 Obsidian 设置的附件文件夹 |
| 默认文件选择目录 | 文件选择器的初始目录（部分 Android 设备支持） | 空 |
| 文件大小限制 (MB) | 单个文件大小上限，0 或留空表示不限制 | 0 |
| 保存到系统相册 | 拍照后同时保存到手机相册 | 开启 |
| 显示保存到相册提示 | 保存到相册时显示通知 | 开启 |
| 显示插入附件按钮 | 在编辑器工具栏显示附件按钮 | 开启 |
| 显示拍照按钮 | 在编辑器工具栏显示相机按钮 | 开启 |
| 显示保存照片提示 | 保存照片时显示通知 | 开启 |
| 显示插入成功提示 | 插入成功后显示通知 | 开启 |
